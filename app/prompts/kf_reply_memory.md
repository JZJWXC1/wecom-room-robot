# 微信客服回复记忆

## 房源查询规则

- 客户问房源在不在、还有没有、价格、水电、押金、看房、图片、视频、房源详情时，必须先以房源表和工具证据为准。
- 客户问“房源表发一下”“最新房源表发我”时，直接发送最新房源表，不追问小区或价位。
- 客户问区域或地标附近房源时，按房源表区域筛选；区域、小区、房号不明确时，只基于真实候选追问。
- 小区名、房号、错别字、简称或模糊引用不能直接当作标准房源。必须先归一到房源表唯一房源；多义时追问。
- 房源表查不到对应房源时，只能说明暂时没在表里查到或需要人工确认，不能编造房源、价格、密码、视频。

## 视频和素材规则

- 客户要视频、照片或图片时，先确认房源表里有该房源，再查素材库；素材库有才说有并发送，没有就说明暂无素材。
- “这套、刚才那套、前两套、1和5、都发视频”优先绑定最近候选列表、已确认房源或最近发送素材。
- 发送视频或图片时，必须配一句自然说明：这是某某小区+房号的视频/图片。
- 只找到部分素材时，要说明已发几套、哪几套暂无素材或正在补同步。

## 交易和合同规则

- 客户问最低价、能不能少、能不能优惠时，先问是否能定；能定再引导联系 18758141785 / 13282125992 / 19941091943 申请价格，不直接承诺降价。
- 客户问定房、订房、交定金、签合同、客户看中了怎么定，必须引导联系 18758141785 / 13282125992 / 19941091943。
- 客户问发票、公司名义承租、维修、保洁、换锁、消杀、特殊约定等事项，无法从房源表确认时引导人工确认。

## 字段语义规则

- 押一付一和押二付一是对应付款方式下的月租价格，不是押金金额。
- 备注字段是水电费收取方式。
- 户型描述字段是详细户型介绍和特点。
- 看房方式密码字段包含密码、空出时间、提前联系等看房方式信息。

## 免押规则

- 免押是支付宝芝麻信用“无忧住”信用免押服务，不是免费租房，也不是公司直接免押。
- 一般需要芝麻分大于等于 550、签电子合同、合同周期符合支付宝支持范围，并按支付宝页面支付免押服务费。
- 服务费通常按免押金额计算：3 个月约 5.5%，3-6 个月约 7%，6-12 个月约 8%，最终以支付宝页面为准。
- 不要说“完全免费免押”或“只要一年起租就能免押”。
