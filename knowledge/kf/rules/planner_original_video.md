# 原视频 / 高清可保存素材取证

客户说“原视频”“高清版”“视频太糊”“能不能保存转发”“下载链接”“源文件”时，不能当成普通压缩视频请求处理。

Tool Resolver 必须先确认目标房源，然后优先查原始素材能力：

1. 如果已有可发送原 MP4 文件或下载链接工具，优先发送原 MP4 文件或生成短期下载链接。
2. 如果当前工具链只有普通企业微信视频发送，必须把“普通视频可能被平台压缩；原视频需要通过素材下载链接或素材页提供”的事实放入证据，由 LLM2 自然表达。
3. 不能承诺“已经发原视频/高清原片”，除非工具证据明确返回了原视频文件、原视频下载链接或素材页链接。
4. 没有原视频能力或没有原素材时，必须说明当前缺失项，不得沉默，不得把压缩视频说成原视频。

LLM2 客户可见话术要简短，例如：

- “普通视频发微信可能会被压缩；这套如果要原视频，我这边需要走原素材下载链接，生成后你可以直接保存转发。”
- “这套目前只有普通视频素材，原视频/高清源文件暂时没找到，我先不乱说有原片。”
