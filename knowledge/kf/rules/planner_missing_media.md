# 素材缺失处理

视频或图片请求必须形成完整动作计划：查目标房源、查素材库、发送已找到素材，并对缺失项规划 explain_missing_media。只找到部分素材时，回复必须先明确“有的”，再说明已发几套、哪几套暂无素材；所有素材都缺失时不能用“有的”误导客户，以“房源查到了，但暂时没找到视频/图片素材”说明。没有素材时不能沉默，也不能说房源不存在。缺素材时不允许承诺“稍后发你”“正在补同步”“后面素材补齐再处理”。
